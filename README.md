# 3D Voxel Model

Pixel art 3D model for my portfolio at [craftaholic.sh](https://craftaholic.sh
)

## Workflow

MagicaVoxel (.vox) → Blender (.blend) → GLB (final)

### 1. MagicaVoxel
Create and paint the voxel model.
- Output: `1. MagicalVoxel/my 3d.vox`

### 2. Blender
Import `.vox` file, adjust materials/lighting if needed.
- Project: `2. Blender/3d-model.blend`

### 3. Export
Export to GLB format for web use.
- Output: `3. FinalArtifact/3d.glb`

## Tools
- [MagicaVoxel](https://ephtracy.github.io/) - Voxel editor
- [Blender](https://www.blender.org/) - 3D modeling & export
